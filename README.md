# Tank-2-Player-Game
🎮 2-player tank battle game playable in any browser — no installs needed
A two-player tank battle game built with vanilla HTML, CSS, and JavaScript. 
No libraries, no installs just open the file in a browser and play.

## Controls
| | Player 1 | Player 2 |
|---|---|---|
| Move | WASD | Arrow Keys |
| Shoot | Space | Enter |

## Features
- Local 2-player on the same keyboard
- Wall maze to fight around
- 3 HP per round with score tracking across rounds
- Particle explosion effects
- CRT scanline aesthetic

## How to play
Download `tank-battle.html`, open it in any browser, and click Start Game.
